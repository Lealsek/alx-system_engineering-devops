# 0x00-shell_basics

This directory contains files related to the basics of shell scripting.

## Contents
- 0-current_working_directory: A script that prints the absolute path name of the current working directory.

## Usage
To use the scripts in this directory, follow these steps:
1. Clone or download the `alx-system_engineering-devops` repository.
2. Navigate to the `0x00-shell_basics` directory.
3. Make the script executable by running `chmod u+x [script_name]`.
4. Run the script with `./[script_name]`.
